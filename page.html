Rolimon's Item Status!

Rolimon's Item Status searches a Roblox limited item with its UAID or serial identifier to see the previous owners to determine if the item is "poisoned" or "clean."

If you are interested in testing the Item Status scanner click the Item Status button!

Item Status Preview

Currently the Item Status is in developement. This is a preview of how it will look like.

Currently you can only see the status of an item by manually using the Item Status checker.

When will Item Status be out?

In a couple weeks when developement is finished.

  • There is currently already a way to search an item's UAID or serial indentifier to see if it's "poisoned" or "clean"
  • We are implementing it into player inventories and the item catalog
  • Your Rolimon's profile page must be viewed in order to trigger a scan of your inventory

Implementation may take longer than expected.
